linker

(type)

A linker is a function that defines the layout of a linked rectangle in a host rectangle.

A linker is a function that defines the layout of a linked rectangle in a host rectangle.
A linker must satisfy these conditions:
  • For a given set of input parameters the a linkers output will always be the same.
  • A linker does not procduce any side effects.
  • A linker returns four numbers defining a rectangle (x, y, w, h).

function

linker(hw, hh, x, y, w, h, xval, yval, minw, maxw, minh, maxh) 

This is the common definition of a linker. Linkers will vary in definition of xval and yval, and of course the result for a given set of input parameters.
