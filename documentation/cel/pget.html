cel:pget

property getter returns properties of a cel by name.

      acel:pget(...)

arguments

  • cel self self
  • string 'x' acel.x
  • string 'y' acel.y
  • string 'w' acel.w
  • string 'h' acel.h
  • string 'xval' acel.xval
  • string 'yval' acel.yval
  • string 'face' acel.face
  • string 'minw' acel.minw
  • string 'maxw' acel.maxw
  • string 'minh' acel.minh
  • string 'maxh' acel.maxh

returns

  • cel self self
  • ... ... the corresponding value for each property name passed in.

notes

pget can be more efficient when you need to get mulitiple properties, but less effecient for 1.

examples

    local cel = require 'cel'
    local host = ...

    local document = cel.document.new()
    local acel = cel.new(100, 100, cel.color.rgb(133, 133, 133))

    document:put(acel)
    document:write(string.format('x, y, w, h = %d, %d, %d, %d', acel:pget('x', 'y', 'w', 'h')))

    document:link(host, 'edges')
