mouse

(type)

A cel mouse represents the mouse inputsource.

A cel mouse represents the mouse inputsource. The cel driver defines values for mouse buttons etc.

buttons

  • left The driver assigned value for the left mouse button.
  • middle The driver assigned value for the middle mouse button.
  • right The driver assigned value for the right mouse button.

states

  • unknown The driver assigned value indicating that Cel does not know if a mouse button is pressed
  • normal The driver assigned value indicating the mouse button is not pressed
  • pressed The driver assigned value indicating the mouse button is pressed

wheeldirection

  • up The driver assigned value for a wheel scrollup.
  • down The driver assigned value for a wheel scrolldown.
