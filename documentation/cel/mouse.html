cel.mouse

(module)

Refers to the mouse inputsource.

Refers to the mouse inputsource.

mouse

(type)

property

mouse.buttons  A table with each entry identifying a unique mouse button.

A table with each entry identifying a unique mouse button. Each entry is assigned a value by the driver.

left
The driver assigned value for the left mouse button.
middle
The driver assigned value for the middle mouse button.
right
The driver assigned value for the right mouse button.

property

mouse.states  A table with each entry identifying a unique mouse button state.

A table with each entry identifying a unique mouse button state. Each entry is assigned a value by the driver.

up
The driver assigned value indicating the mouse button is not pressed
down
The driver assigned value indicating the mouse button is pressed

property

mouse.wheel  A table with each entry identifying a mouse wheel scroll direction.

A table with each entry identifying a mouse wheel scroll direction. Each entry is assigned a value by the driver.

up
The driver assigned value for a wheel scrollup.
down
The driver assigned value for a wheel scrolldown.

Returns the x and y position of the mouse cursor relative to the root cel.

returns
number
x position of mouse cursor.
number
y position of mouse cursor.

function

mouse:vector()  Returns the difference of the current mouse cursor position and the previous position.

Returns the difference of the current mouse cursor position and the previous position.

returns
number
x component of vector.
number
y component of vector.

function

mouse:isdown(button)  Returns true if specified button is down.

Returns true if specified button is down.

arguments
button any
a value present in the mouse.buttons table.
returns
boolean
true if the button is down, else false.

function

mouse:pick()  Forces picking algorithm that determines which cel(s) the mouse is in to run.

Forces picking algorithm that determines which cel(s) the mouse is in to run. This is normally only run when the mouse cursor is moved or a button state changes. It is too costly to always run when a cel moves, but can be necessary in special cases.

returns
mouse
